# Ninart Vision — Brand assets

Use **only** files in this folder. Do not mix Font Awesome or legacy PNG paths for brand UI.

## Core identity

| Asset | Path | Usage |
|-------|------|--------|
| Official logo | `logo.webp` | Navbar, footer, loading screen, Open Graph / Twitter, JSON-LD `logo` |
| Favicon (legacy PNG) | `favicon.png` | Fallback `<link rel="icon">`, Apple touch icon |
| WebP favicon | `logo.webp` | Modern browsers via `<link rel="icon" type="image/webp">` |
| Footer atmosphere | `earth.webp` | Footer + service hero backgrounds |
